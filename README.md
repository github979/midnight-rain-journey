# 🌧️ Midnight Rain

A cinematic, mood-driven web project inspired by Taylor Swift’s *Midnight Rain* — built around motion, sound, and subtle interaction.

---

## 🌐 Live
[(Midnight rain)](https://github979.github.io/midnight-rain-journey/)

---

## 🎨 Design Language

Minimal, dark, and slightly dramatic.  
Typography and motion do most of the work — not heavy UI.

---

## 🔤 Typography

- **Cormorant Garamond** — editorial, soft, expressive  
- **Cinzel** — structured, classical presence  
- **DM Mono** — contrast, precision
