# 🌧️ Midnight Rain Journey

> *"I was sunshine. I chose midnight rain."*

A personal website inspired by the Taylor Swift Eras Tour aesthetic — dark, cinematic, poetic.

---

## 📁 Project Structure

```
midnight-rain/
├── index.html          # Main HTML — semantic, clean
├── css/
│   └── style.css       # All styles, tokens, animations
├── js/
│   └── main.js         # Rain canvas, cursor, music player, scroll FX
└── assets/
    ├── music.mp3       # Background music (replace with your track)
    └── eras-badge.svg  # Custom Eras Tour inspired badge
```

---

## ✨ Features

- 🌧️ **Animated rain canvas** — live falling rain in background
- 🎵 **Background music player** — floating vinyl-style player, plays on first click
- ✦ **Custom gold cursor** — dot + trailing ring
- 🌙 **Scroll-triggered reveals** — journal entries fade in on scroll
- 🏆 **Eras Tour badge** — slowly spinning SVG badge (custom made)
- 📱 **Fully responsive** — works on mobile
- 🎨 **Cinematic dark theme** — Cormorant Garamond + DM Mono + Cinzel fonts

---

## 🚀 Deploy to GitHub Pages

1. Push this folder to your GitHub repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your site will be live at `https://yourusername.github.io/midnight-rain-journey/`

---

## 🎵 Change the Music

Replace `assets/music.mp3` with any `.mp3` file of your choice.
The player auto-detects it. Keep the filename `music.mp3` or update the `<source src="...">` in `index.html`.
